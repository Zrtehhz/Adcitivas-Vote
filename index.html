Plugin de vote

Bienvenue sur le plugin de vote Top-Serveurs !

Ce plugin vous permet de recevoir des votes directement sur votre serveur FiveM/RedM.

Comment ça marche ?

  1. Installez le plugin sur votre serveur
  2. Configurez le avec votre token
  3. Ecoutez l'événement onPlayerVote pour récompenser vos joueurs
  4. Activez le plugin sur le panel d'administration du site

Pour plus d'informations, consultez la documentation.
